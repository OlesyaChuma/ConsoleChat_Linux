# ConsoleChat-Linux

Консольный чат на C++ для Linux.

## Описание проекта

Программа позволяет:

- регистрировать пользователей;
- выполнять вход по логину и паролю;
- отправлять личные сообщения;
- отправлять сообщения всем пользователям;
- сохранять зарегистрированных пользователей в файл;
- сохранять историю сообщений в файл;
- восстанавливать состояние программы после повторного запуска.

Проект выполнен с использованием библиотеки `std::filesystem`.

## Структура хранения данных

Данные сохраняются в каталог `data/`:

- `data/users.txt` — данные пользователей;
- `data/messages.txt` — история сообщений.

## Защита файлов

По ТЗ файлы должны быть недоступны другим пользователям Linux.

Для этого в программе автоматически задаются права доступа:

- для каталога `data` — `700`
- для файлов `users.txt` и `messages.txt` — `600`

Это значит:

- только владелец может читать, изменять файлы и работать с каталогом;
- остальные пользователи доступа не имеют.
